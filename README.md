# Product Stock API

This app demonstrates how to build a Rest APIs using Nest.js, Mongoose, Swagger, Passport etc.

## Features

- User resource
- Authentication (Local & JWT Strategies)
- Product resource
- Product Stock resource
- Purchase Order resource
- Payment resource
- Pagination for findAll method
- Some tests sample

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
